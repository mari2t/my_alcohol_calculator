## Purpose of creation

1. 複数のお酒を飲んだ時のアルコール量を確認したい
2. 1週間の目標をどのくらい達成できたか確認したい
3. 1週間にあと何gアルコールを摂取できるか確認したい
4. Django のアプリをつくってみたい

## Specification

1. 1日のアルコール総量を計算できる
2. 1日に複数の飲んだお酒を登録することができる
3. 飲んだお酒は編集、削除ができる
4. 1週間のアルコール量の目標を入力できる
5. 1週間のアルコール量の目標の達成具合が表示できる
